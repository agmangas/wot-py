Objects defined in the Scripting API specification represented as classes that are basically dict-wrappers.

wotpy.wot.dictionaries.base Base class for WoT dictionaries.
wotpy.wot.dictionaries.filter Wrapper class for dictionaries to represent Thing filters.
wotpy.wot.dictionaries.interaction Wrapper classes for dictionaries for interaction initialization that are defined in the Scripting API.
wotpy.wot.dictionaries.link Wrapper classes for link dictionaries defined in the Scripting API.
wotpy.wot.dictionaries.schema Wrapper classes for data schema dictionaries defined in the Scripting API.
wotpy.wot.dictionaries.security Wrapper classes for security dictionaries defined in the Scripting API.
wotpy.wot.dictionaries.thing Wrapper class for dictionaries to represent Things.
wotpy.wot.dictionaries.version Wrapper classes for versioning dictionaries defined in the Scripting API.
